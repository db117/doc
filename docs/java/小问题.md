## properties配置文件中的换行(多行)的坑

properties中都是以name=value这样的k-v字符串对形式保存的。
在写properties文件时,如果value非常长,看起来是非常不方便的，可以用\来换行(最后一行不需要\)，如下 :

sonar.exclude=a.java \
                     	  b.java \
                     
这里的坑就是\必须是每行的**最后一个字符**！
