#### 替换为`huggingface`镜像站

进入安装目录

使用`vsCode`等IDE打开目录，然后把里面的`huggingface.co`链接替换为`hf-mirror.com`

```
# windows 默认位置
C:\Users\user\AppData\Local\LM-Studio\app-x.x.x
```
